# Firefox 初期設定

### ブックマークをタブで開く

```
browser.tabs.loadBookmarksInTabs
```
`true` に設定


### ロケーションバーからタブで開く

```
browser.urlbar.openintab
```
`true` に設定

### 検索ボックスからタブで開く

```
browser.search.openintab
```
`true` に設定

### JavaScriptによって新規ウィンドウで開くリンクをタブで開く

```
browser.link.open_newwindow.restriction
```
`0` に設定
