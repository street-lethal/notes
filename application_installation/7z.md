# 7z

## インストール

https://www.7-zip.org/download.html から 64-bit Linux x86-64 をダウンロードして解凍

```shell
cd {解凍されたディレクトリ}
```

```shell
sudo cp 7zzs /usr/local/bin/
sudo ln -s /usr/local/bin/7zzs /usr/local/bin/7z
```

## 圧縮
```shell
7z a -tzip -p -mem=AES256 OUTPUT_FILE.zip UNZIPPED_DIRECTORY
```

## 解凍
```shell
7z x ZIPPED_FILE.zip
```

## 暗号化方式確認
```shell
7z l -slt ZIPPED_FILE
```
